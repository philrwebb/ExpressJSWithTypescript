# This project is the template for code generation of a typescript crud api

## See https://github.com/philrwebb/generation

In this project, the code in src/schemas, src/routes, src/types/model is generated
The code in src/handlers, src/middleware, src/repositories and src/index.ts is fixed template code

The repositories contains a generic interface and two implementations (FileRepository and InMemoryRepository). Current generation is to the FileRepository implementation.
